How to Use

Basic Features

Click any item to mark it as completed

Ctrl+Click (Cmd+Click on Mac) to mark an item as needing attention

Click section headers to collapse/expand sections

Search Features

Type any text to search item names. Use the following tags to search for specific entries.

You can negate a search term by prefixing it with a !.

Tag Description
@done Entries marked as complete
@inprogress Entries marked as in progress
@attention Entries marked as needing attention
@tags Entries with tags
@notes Entries with notes

Tag Search

Use a # followed by the tag name to search for items with that tag.
This works with partial matches as well.

Notes System

Click the ‚Ñπ icon on any item to add notes

Notes can be categorized as Bug, Note, Todo, or Question

Notes are saved automatically when closing the modal. Empty notes are removed.

Import/Export

Use Export to save your progress

Use Import to restore a previously saved state
